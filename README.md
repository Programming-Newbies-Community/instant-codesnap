# 📸 瞬时码窗 · Instant CodeSnap

> 手机优先的代码片段即时分享工具  
> 专门解决“发代码给朋友/同事，格式全乱”的痛点

## ✨ 特性

- 📱 **手机优先** - 专门为手机屏幕优化，大按钮、大输入区
- ⚡ **即时分享** - 输入代码 → 生成链接 → 发给朋友
- 🎨 **语法高亮** - 支持 JavaScript/Python/HTML/CSS/Java/C++ 等20+语言
- ⏰ **自动销毁** - 可选1小时/24小时/7天后自动删除，保护隐私
- 🔗 **永久链接** - 生成的链接可直接访问，无需登录
- 🌙 **深色模式** - 保护眼睛，适合程序员


## 📦 技术栈

- 前端：原生 HTML/CSS/JavaScript（无框架，极致轻量）
- 存储：JSONBin.io（免费，无需服务器）
- 代码高亮：手动实现，保持轻量

## 🔧 本地运行

1. 克隆仓库
2. 去 [JSONBin.io](https://jsonbin.io) 注册免费账号
3. 创建一个 Bin，获取 `BIN_ID` 和 `API_KEY`
4. 修改 `script.js` 中的配置
5. 用手机浏览器打开 `index.html`
